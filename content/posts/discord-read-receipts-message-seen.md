# Does Discord Have Read Receipts? (2026 Update)

## The Short Answer: Sort of
Discord does not have traditional read receipts like some other messaging platforms, but it does display a "read" status when a user views a message, indicated by the user's avatar appearing next to the message. However, this feature is limited to direct messages and group chats with fewer than 10 participants, and it does not provide a timestamp or explicit "read" indicator.

## How to Use Read Receipts in Discord (If Yes)
Since Discord's read status is not a traditional read receipt feature, there is no specific setup or configuration required to use it. However, here's how you can observe the read status:
1. Navigate to a direct message or group chat with fewer than 10 participants.
2. Send a message to the chat.
3. Observe the user avatars that appear next to the message as other users view it.
